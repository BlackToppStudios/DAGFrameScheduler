Detailed Description

template<typename RecordType, typename MathType>
class Mezzanine::WeightedRollingAverage< RecordType, MathType >

A weighted average that does math with every insertion and stores nothing.

This is possibly an optimization over the BufferedRollingAverage, It has not yet been shown whether the amount of memory for each task will significantly affect load times (From RAM to CPU cache), when starting a rolling average, or whether any innacuracies in sorting using this would outweigh that.
