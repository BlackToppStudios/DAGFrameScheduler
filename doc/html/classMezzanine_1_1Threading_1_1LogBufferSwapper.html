Detailed Description

Swaps all of the Logs so the usable logs can be commited, and the usable commitable.

Note
If other doublebuffered resources are used then this or another work unit like it should be adjusted/created to swap those buffers.

This needs to have every other workunit that uses the logger as a dependency. If you use the log in a work unit that work unit needs to depend on this one, or this one needs to depend on that one.
